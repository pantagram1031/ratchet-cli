# ratchet-cli

Deterministic ratchet for AI coding agents — a CLI implementation of Park Jun-woo's
**Reins Engineering** and **Ratchet Pattern**.

> Full README lands in Phase 4. This file is a stub; the package builds and `ratchet`
> works regardless.

## Install (preview)

```bash
pipx install git+https://github.com/pantagram1031/ratchet-cli
ratchet init
ratchet add "first item"
ratchet next
ratchet submit
```
